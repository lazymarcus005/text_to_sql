# Database Schema Context for Text-to-SQL

รายการตาราง (Collection name) แบ่งตามประเภทการใช้งาน เพื่อใช้เป็นความรู้ประกอบในการสร้างคำสั่ง SQL:

### 1. Configuration

* `roles`: ข้อมูลบทบาทและสิทธิ์ผู้ใช้งาน
* `users`: ข้อมูลผู้ใช้งานในระบบ
* `audit_log`: บันทึกประวัติการใช้งานและการเปลี่ยนแปลง
* `departments`: ข้อมูลหน่วยงาน (โครงสร้างแบบ Tree)
* `departments_provinces`: แผนผังความเชื่อมโยงหน่วยงานรายจังหวัด
* `departments_livestock`: แผนผังความเชื่อมโยงหน่วยงานด้านปศุสัตว์
* `departments_amphur`: แผนผังความเชื่อมโยงหน่วยงานรายอำเภอ
* `webpublic`: การตั้งค่าข้อมูลเว็บไซต์สาธารณะ
* `webpublic_files`: ข้อมูลคอลเลกชันไฟล์สำหรับเว็บไซต์สาธารณะ
* `sd01_sign_digital`: พื้นที่สำหรับจัดเก็บเอกสารลงนามดิจิทัล
* `sd02_sign_digital`: ข้อมูลการเก็บเอกสารลงนามดิจิทัล
* `bd17_map_m01`: ข้อมูลแผนผังเชื่อมโยงตาราง M01
* `bd17_map_m02`: ข้อมูลแผนผังเชื่อมโยงตาราง M02

### 2. External

* `ex_farmer`: ข้อมูลเกษตรกรจากระบบ E-Regis
* `ex_animal_amount`: ข้อมูลจำนวนสัตว์จากระบบ E-Regis
* `ex_farmer_animal_amount`: ข้อมูลจำนวนสัตว์รายบุคคล (One-to-one)

### 3. Master 

* `livestock`: ข้อมูลเขตปศุสัตว์
* `m_province`: ข้อมูลจังหวัด
* `m_amphur`: ข้อมูลอำเภอ
* `m_tambon`: ข้อมูลตำบล
* `m_village`: ข้อมูลหมู่บ้าน
* `m_animal_type`: ข้อมูลประเภทสัตว์
* `m_feed_type`: ข้อมูลประเภทอาหารสัตว์
* `m_healthcare_type`: ข้อมูลประเภทการรักษาสัตว์
* `m_disaster_type`: ข้อมูลประเภทภัยพิบัติ
* `m_anima_sub_type`: ข้อมูลประเภทสัตว์ย่อย
* `livestock_zone`: ข้อมูลสำนักงานปศุสัตว์
* `m_status`: ข้อมูลสถานะต่างๆ ในระบบ
* `m101_animal_stock`: ข้อมูลสต็อกสัตว์หลัก
* `m102_dms`: คอลเลกชันไฟล์สำหรับระบบจัดการเอกสาร (DMS)

### 4. Preparation 

* `migration_area`: ข้อมูลจุดอพยพสัตว์
* `vehicle`: ข้อมูลยานพาหนะช่วยเหลือ
* `animal_pen`: ข้อมูลคอกสัตว์
* `grass_supply`: ข้อมูลเสบียงหญ้าสำรอง
* `emergency_kit`: ข้อมูลถุงยังชีพสัตว์
* `veterinary_unit`: ข้อมูลหน่วยสัตวแพทย์เคลื่อนที่
* `centering_command`: ข้อมูลคำสั่งจัดตั้งศูนย์อำนวยการ

### 5. Transaction (ข้อมูลรายการและบันทึกเหตุการณ์ภัยพิบัติ)

* `b000_open_disaster`: ข้อมูลประกาศพื้นที่ภัยพิบัติ
* `b100_disaster_area`: ข้อมูลประกาศเขตการช่วยเหลือ
* `doctor`: ข้อมูลรายชื่อหมอ/สัตวแพทย์
* `collection_document`: ข้อมูลการรวบรวมเอกสารต่างๆ
* `tbl_sps01`: รายงาน ศปส.1 (ยืนยันจำนวนสัตว์รายพื้นที่)
* `tbl_sps02_round_report`: รอบรายงานของ ศปส.2
* `tbl_sps02_help_log`: ประวัติการบันทึกการช่วยเหลือ (ศปส.2)
* `tbl_sps02`: รายงาน ศปส.2 (สรุปการช่วยเหลือรายวัน)
* `tbl_sps02_help_summary`: สรุปยอดการช่วยเหลือสะสม
* `tbl_sps03`: รายงาน ศปส.3 (สรุปความเสียหายรายวัน)
* `tbl_sps03_round_report`: รอบรายงานของ ศปส.3
* `tbl_sps03_damage_log`: ประวัติการบันทึกสรุปความเสียหาย (ศปส.3)
* `b10ex01`: ข้อมูลการดึงข้อมูลเกษตรกรเพื่อเชื่อมโยงภัย
* `b20_init_help`: ข้อมูลการช่วยเหลือเบื้องต้น
* `b21_feed_count`: ข้อมูลจำนวนอาหารสัตว์ที่แจกจ่าย
* `b22_move_animal`: ข้อมูลการอพยพเคลื่อนย้ายสัตว์
* `b23_healthcare_count`: ข้อมูลจำนวนการรักษาสัตว์
* `b30_init_damage`: ข้อมูลการรายงานความเสียหายในเบื้องต้น
* `b31_damage_count`: ข้อมูลสรุปจำนวนสัตว์ที่เสียหาย
* `b40_help_quata_area`: ข้อมูลโควต้าการช่วยเหลือตามพื้นที่
* `b41_assis_rate_area`: ข้อมูลอัตราการให้ความช่วยเหลือ
* `c10_request_for_relief`: แบบขอรับการช่วยเหลือ (กษ.01)
* `c101_request_area`: ข้อมูลพื้นที่ขอรับการช่วยเหลือ (กษ.02)
* `c102_request_for_relief`: รายละเอียดการขอรับการช่วยเหลือ (กษ.02)
* `c103_request_documents`: ไฟล์เอกสารประกอบคำขอช่วยเหลือ (กษ.02-03)
* `c11_request_damage_count`: จำนวนความเสียหายที่ระบุในคำขอ
* `c12_summary_assis`: ข้อมูลสรุปการช่วยเหลือ (ศปส.4)
* `log_migration_area`: บันทึกประวัติการใช้งานพื้นที่อพยพ
* `report_schedules`: กำหนดเวลาการออกรายงานในระบบ
* `report_schedule_items`: รายละเอียดรายการในกำหนดเวลาการรายงาน
* `documents`: ข้อมูลโครงสร้างแฟ้มเอกสาร (Tree collection)
* `file_information`: ข้อมูลรายละเอียดและ metadata ของไฟล์
* `file_store`: คอลเลกชันสำหรับจัดเก็บไฟล์จริง
* `document_scope`: ข้อมูลขอบเขตการเข้าถึงเอกสาร
* `UsedAnimalStock`: ข้อมูลประวัติการใช้งานเสบียงหญ้าและสต็อกสัตว์
* `d01_tobe_process_files`: ไฟล์เอกสารรอดำเนินการ (ก่อนเกิดภัย)
* `d0101_tobe_process_files`: รายการไฟล์สำหรับรายงาน ศปส.1
* `d0102_tobe_process_files`: รายการไฟล์สำหรับรายงาน ศปส.2
* `d0103_tobe_process_files`: รายการไฟล์สำหรับรายงาน ศปส.3
* `d02_asis_process_files`: ไฟล์เอกสารที่บันทึกผล (หลังเกิดภัย)
* `d03_doc_keywords`: ข้อมูลคีย์เวิร์ดสำหรับค้นหาเอกสาร
* `bd173_files`: ไฟล์เอกสารในระบบ BD17
